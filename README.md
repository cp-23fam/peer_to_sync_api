# Peer to Sync API

API du projet Peer to Sync

Le projet principal est trouvable [ici](https://github.com/cp-23fam/peer_to_sync)

## Prérequis

### Installation

1. [Installer](https://www.mongodb.com/try/download/community) MongoDB localement
2. Mettre en place les variables d'environnement requises
3. `npm i`
4. `npm run start`

### ENV

- **CONNECTION_STRING** (requis) : String de connexion pour la Base de donnée Mongo
- **JWT_PRIVATE_KEY** (requis) : Clé privé pour l'encryption des tokens
